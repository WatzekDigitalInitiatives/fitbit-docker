This is the fitbit app made by the Digital Initiatives at Watzek Library, Lewis & Clark College.

## Development

The application runs on [docker](https://www.docker.com/) containers.

It runs on two containers:

*Web*
```
Ruby -v 2.3.1
Rails -v 4.2.0
```
*Database*
```
PostgreSQL -v 9.5.4
```
